## Introduction

*servers* is the servers of [lspserver](https://github.com/lspserver).



## Docker

```bash
git clone https://github.com/lspserver/servers.git

cd servers
docker build --no-cache -f Dockerfile -t ghcr.io/lspserver/servers:latest .
```
